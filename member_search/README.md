# 회원사 뉴스 검색

## 🚀 설명
memberlist 엑셀 파일 C열 '회원사명' 목록의 기업 뉴스를 검색해서 html 테이블로 출력 (시작~종료날짜 입력)

## 🛠️ 기술 스택
- Python 3.7
- HTML5

## 🎯 설치 및 실행 방법
1. Python 라이브러리 설치:
   `python -m pip install openpyxl requests beautifulsoup4 lxml`
2. 엑셀 파일의 C열의 회원사 목록 확인:
   `memberlist.xlsx`
3. 파이썬 스크립트 실행:
   `newsletter_2.py`
4. 출력된 html 확인:
   `member_news.html`
