# 국가R&D통합공고 확인

## 🚀 설명
국가R&D통합공고에서 현재 날짜 기준 7일 이상 기한이 남은 '산업통상부', '과학기술정보통신부', '중소벤처기업부'의 본공고를 확인, HTML 테이블로 출력하는 간단한 파이썬 코드 입니다.

## 🛠️ 기술 스택
- Python 3.7
- HTML5

## 🎯 설치 및 실행 방법
1. Python 라이브러리 설치:
   `pip install pandas openpyxl selenium webdriver-manager`
   `pip install xlrd openpyxl`
3. 파이썬 스크립트 실행:
   `ntis_limitless.py`
4. 출력된 html 확인:
   `ntis_projects.html`
